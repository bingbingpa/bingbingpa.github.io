  - gatsby-config.js 정리
    - gatsby-plugin-sass 추가하고, gatsby-plugin-styled-jsx, gatsby-plugin-styled-jsx-postcss 2개는 주석처리
  - theme.yaml 삭제 및 관련 의존성 삭제. style 관리는 styled-jsx 로 하지 않고 전역 scss 에서 관리
